_# SmartSpec Workflow: Docs Publisher

**Workflow:** `/smartspec_docs_publisher`  
**Version:** 6.1.1

## 1. Overview

The Docs Publisher takes the documentation generated by `/smartspec_docs_generator` and publishes it to a target platform, such as a static site generator (e.g., Docusaurus), a knowledge base (e.g., Confluence), or a Git repository.

## 2. Key Features

- **Multi-Platform Publishing:** Supports various targets for documentation.
- **Automated Deployment:** Integrates with CI/CD to automate the publishing process.
- **Version-Aware:** Can publish docs for a specific version of the project.

## 3. Usage

```bash
/smartspec_docs_publisher   --source-dir docs/guide   --target-platform docusaurus   --version v1.2.3   --apply
```
