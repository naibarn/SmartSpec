## Technical Details

### Syntax Migration
- **Old syntax:** `--kilocode`
- **New syntax:** `--platform kilo`
- **Reason:** More explicit and consistent with multi-platform support

### Parameter Documentation Structure

Each workflow entry includes:
1. **Description:** Brief explanation of workflow purpose
2. **Version:** Current workflow version
3. **Parameters Table:**
   - Parameter name
   - Status (Required/Optional)
   - Description
4. **Universal Flags:** List of supported universal flags
5. **Usage Examples:** Concrete command examples
